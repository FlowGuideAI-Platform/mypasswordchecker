## ⚠️ Important Legal & Compliance Notes

### 1. FTC Affiliate Disclosure
**Required:** You must disclose affiliate relationships clearly.

**Current Footer (✅ Compliant):**
```html
<p style="margin-top: 0.5rem; font-size: 0.8rem;">
  Affiliate Disclosure: We may earn commissions from recommended products at no extra cost to you.
</p>
```

**Recommendation:** Keep this exact disclosure. It's compliant with FTC guidelines.

### 2. GDPR Compliance (Ad Networks)
- AdSense automatically handles GDPR consent banners
- If using Ezoic, they provide GDPR consent management
- Your privacy policy already covers affiliate links

### 3. Rel Attributes for SEO
Always use these attributes on affiliate links:
```html
<a href="affiliate-url" target="_blank" rel="noopener sponsored">Link Text</a>
```

- `rel="noopener"`: Security best practice
- `rel="sponsored"`: Tells Google it's a paid link (avoids penalties)

### 1. Strategic Link Placement
- Put highest-paying affiliates (1Password, NordVPN) first in the list
- Use specific anchor text: "Get 1Password" converts better than just "1Password"
- Add affiliate links in password feedback section (contextual placement = higher conversions)

### 2. Seasonal Promotions
- Black Friday / Cyber Monday: Password managers run 50% off deals (huge conversion boost)

- Tax season: Promote password managers for securing financial data

### 3. Content Ideas That Drive Affiliate Sales
- "How I Chose My Password Manager" (personal story format)
- "YubiKey Setup Guide" (drives hardware affiliate sales)
- "Password Manager Comparison Chart" (comparison tables convert well)

### 4. Email Capture Strategy
Consider adding email capture at end of password check:

Then send weekly emails with:
- Security tips (builds trust)

- New password threats (keeps engagement high)
